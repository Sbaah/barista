# Barista

Barista is a CLI that can assist you while you build your Java application.

## Installation

Clone this repository:

```bash
git clone https://github.com/wasthishelpful/barista
```

Then compile Barista.java:

```bash
javac Barista.java
```

## Usage

Run Barista:

```bash
java Barista.java
```
